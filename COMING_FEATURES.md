# Kommende Features

Hier werden geplante Funktionen gesammelt, die in Zukunft zur App hinzugefügt werden sollen.

---

### Festivals & Events

- [ ] 🎵 **[S] Playlist-Link:** Festival-Ersteller sollen einen Link zu einer Playlist (z.B. Spotify, YouTube) bei einem Festival hinterlegen können, damit Teilnehmer die Musik des Events vorab entdecken können.
- [ ] 📸 **[L] Festival-Fotos:** Nutzer sollen Fotos von Festivals hochladen und mit anderen Teilnehmern teilen können. Erinnerungen können so direkt beim Event festgehalten und für alle sichtbar gemacht werden.
- [ ] 🔒 **[L] Gruppen-exklusive Festivals:** Festival-Ersteller sollen Festivals optional nur für Mitglieder bestimmter Gruppen zugänglich machen können. Damit lassen sich private Events gezielt für ausgewählte Personenkreise organisieren.

### Gruppen

- [ ] 💬 **[M] Kommentare in Gruppen:** Gruppenmitglieder sollen Kommentare in einer Gruppe hinterlassen können, um den Austausch innerhalb der Gruppe zu fördern.
- [ ] 👥 **[M] Gruppen nur für Freunde:** Gruppenbesitzer sollen einstellen können, dass nur Freunde einer Gruppe beitreten dürfen. Das gibt mehr Kontrolle darüber, wer Teil einer Gruppe werden kann.

### Nutzer & Soziales

- [ ] 📅 **[M] Verfügbarkeits-Kalender:** Nutzer sollen in ihrem Profil Zeiträume markieren können, in denen sie für Festivals verfügbar sind. Freunde können so auf einen Blick sehen, wann gemeinsame Events möglich wären.
